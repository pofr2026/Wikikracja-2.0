{% trans "How do tasks work?" %}
{% trans "Mine" %}

{% blocktrans %}The tab collects tasks that directly concern you: tasks that you have undertaken (you are the owner) and tasks in which you have declared your help (you have clicked "I want to help").{% endblocktrans %}

{% trans "Pending" %}

{% blocktrans %}Active tasks that do not yet have an owner or have an owner but have not gathered enough support (the difference in votes "I want to help" minus "we are not doing this" is less than 2). Anyone can take such a task using the "Take" button.{% endblocktrans %}

{% trans "In progress" %}

{% blocktrans %}Tasks that have an owner and have collected at least 2 net declarations of help (votes "I want to help" minus votes against ≥ 2). The owner can resign - the task then returns to the waiting list. Votes ≤ −2 automatically reject the task.{% endblocktrans %}

{% trans "Completed" %}

{% blocktrans %}Tasks closed by the owner (completed or canceled) and tasks rejected by the community (total votes ≤ −2). A completed task can be reopened.{% endblocktrans %}

{% trans "Declaration of aid" %}

{% blocktrans %}The "I want to help" button is a declaration of commitment - not a like. The number of people who clicked it is visible on the task tile. Sufficient support (≥ 2 net) is a condition for the task to be implemented.{% endblocktrans %}
